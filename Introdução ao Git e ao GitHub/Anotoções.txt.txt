O que é o Git?
 É um sistema de controle de versão ultilizado atualmente pela grande maioria dos desenvolvedores. Com ele podemos criar todo histórico de alteração no código do nosso projeto e falcimente voltar para qualquer ponto para saber como código estava naquela data. 

* Desenvolvido em 2005 por Linus Torvalds, o famoso criador do kernel do sistema operacional Linux.

O que é Git/Hub?
 É um sistema baseado em nuvem que hospeda um sistema de controle de versão chamado Git. Ele permite que os desenvolvedores  colaborem e façam mudanças em projetos compartilhados enquanto mantém um registro detalhado de seu progresso.

* Desenvolvido em 2008 pelos Fundadores: Tom Preston-Werner, Chris Wanstrath, Scott Chacon, P. J. Hyett

* Link como usar o GitHub:https://tecnoblog.net/responde/como-usar-o-github-guia-para-iniciantes/
